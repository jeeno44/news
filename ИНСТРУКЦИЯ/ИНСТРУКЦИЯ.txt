    Инструкция по разворачиванию проекта.

ВАЖНО !!! - Проект был написан из под Linux.

0) Предварительные настройки.
    а) У вас должен быть установлен git
    б) У вас должен быть установлен composer
    в) У вас должен быть установлен сервер баз данных Mysql (не ниже версии 5.6, желательно версия 5.7)
    г) У вас должен быть установлен php сервер версии не ниже 7.0 желательно версия 7.2
    д) У вас должны быть навыки навигации в командной строке (linux terminal, либо cmd windows)

1) С помощью командной строки зайдите в директорию в которой будет располагаться проект
    а) Выполните команду https://github.com/jeeno44/news.git (проект будет скопипрован вам в папку)
    б) Дождитесь окончания клонирования удалённого репозитория
    в) Зайдите в директорию news
    г) Выполните команду composer update (менеджер зависимостей обновит все нужные пакеты для заработки)
    д) Если во время выполнения команды composer update у вас появятся ошибки, то скорее всего у вас не установлены php библиотеки
        необходимые для этого фреймворка. Установите эти библиотеки и снова запустите команду composer update
    е) Если выполнение команды composer update прошло успешно, запустите следующую команду - "php artisav -v"
        Если установка прошла успешно, вы увидите список доступных команд фреймворка Laravel.

2) Зайдите на сервер mysql и выполните следующий запрос sql - "CREATE DATABASE news CHARACTER SET utf8 COLLATE utf8_general_ci;" (БЕЗ КАВЫЧЕК)

3) Проверьте в корневой директории проекта наличие файла с перемеными окружения ".env"

4) Проверьте настройки базы данных в этом файле, они должны быть такими:
        DB_CONNECTION=mysql
        DB_HOST=127.0.0.1
        DB_PORT=3306
        DB_DATABASE=news        (Имя базы данных которую вы создали на сервере)
        DB_USERNAME=user        (Логин пользователя сервера Mysql)
        DB_PASSWORD=password    (Пароль пользователя сервера Mysql)

5) Из корневой директории проекта выполните миграции для добавления таблиц проекта, командой - "php artisan migrate"

6) Из корневой директории проекта выполните посев данных, командой - "php artisan db:seed"

7) Из корневой директории проекта запустите php сервер командой - "php artisan serve"

8) Откройте в браузере адрес сайта - "http://127.0.0.1:8000"

9) Незарегистрировнным пользователям доступны новости, разделённые по рубрикам.

10) Авторизируйтесь на сайте как администратор. Логин - root@mail.ru Пароль 123456.
    Либо как редактор.
        Логины редакторов и пользователей доступны в файле посева по адресу - "Project/database/seeds/AppendAdminAndEditors"
        Пароль для всех пользователей (включая админа) - 123456.
    Либо зарегистрируйтесь как новый пользователь.

11) В таблице users созданы несколько пользователей и ректоров.
    Также добавлены несколько новостей для пробного запуска.
    Логины редакторов и пользователей доступны в файле посева по адресу - "Project/database/seeds/AppendAdminAndEditors"
